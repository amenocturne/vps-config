# Эффективное растягивание формул горячими клавишами

## Задача
Нужно заполнить формулу на сотни столбцов (например, от A до BA = 53 столбца).
Перетаскивание мышью — медленно и чревато ошибками.

## Решение

### Способ 1: Заполнение вправо (самый быстрый)
1. Выдели ячейку с формулой
2. Поставь ограничитель в последнем целевом столбце (или используй Ctrl+End)
3. `Ctrl+Shift+Вправо` — выделяет от текущей ячейки до ограничителя
4. `Ctrl+R` — заполняет формулу вправо по выделению

### Способ 2: Копирование + Специальная вставка
1. Скопируй ячейку с формулой (`Ctrl+C`)
2. Выдели целевой диапазон (`Ctrl+Shift+Вправо`)
3. Специальная вставка → Только формулы (чтобы не копировать форматирование)

### Чего НЕ делать
- Не перетаскивай маркер заполнения мышью/тачпадом
- Не используй Shift+Стрелка для расширения выделения по одной ячейке
- Не используй Ctrl+V (тащит за собой форматирование)

### Фиксация ссылок при растягивании
F4 для переключения режимов фиксации:
- `$A$1` — полная фиксация (ничего не сдвигается)
- `A$1` — фиксация строки (столбец сдвигается при растягивании вправо)
- `$A1` — фиксация столбца (строка сдвигается при растягивании вниз)
- `A1` — без фиксации (сдвигается всё)

## Ключевой инсайт
Трюк с ограничителем (значение в последнем столбце) даёт Ctrl+Shift+Вправо
точку остановки. Без него выделение улетает в конец листа.
Альтернатива — скрыть столбцы за пределами диапазона через Ctrl+).
